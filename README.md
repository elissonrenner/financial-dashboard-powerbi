# financial-dashboard-powerbi
Power BI dashboard for analyzing financial performance, including revenue, expenses, and profitability metrics.

# 📊 Dashboard Financeiro - Power BI

Este projeto apresenta um dashboard financeiro desenvolvido no Power BI com o objetivo de analisar indicadores essenciais de desempenho financeiro e apoiar a tomada de decisões estratégicas.

## 🎯 Objetivo

O dashboard foi criado para fornecer uma visão clara e organizada sobre a saúde financeira de um negócio, permitindo acompanhar receitas, despesas e lucratividade ao longo do tempo.

## 📈 Principais Indicadores

- Receita total
- Despesas totais
- Lucro líquido
- Análise de lucratividade
- Evolução financeira ao longo do tempo

## 🧠 Possíveis Insights

- Identificação de períodos com maior lucro ou prejuízo
- Análise da relação entre receitas e despesas
- Avaliação da evolução financeira ao longo do tempo
- Apoio à tomada de decisão com base em indicadores financeiros

## 🛠️ Ferramentas Utilizadas

- Power BI
- Microsoft Excel
- Técnicas de análise e visualização de dados

## 🗂️ Base de Dados

Os dados utilizados neste projeto são **fictícios**, criados exclusivamente para fins de estudo e desenvolvimento de portfólio.
